Landing Page - Prompts Fotos Profissionais com IA

Esta é uma landing page de alta conversão desenvolvida para a estrategista digital Bianca Oliveira. O objetivo da página é a venda de um infoproduto focado em prompts de inteligência artificial para criação de fotografias profissionais.

🚀 Tecnologias Utilizadas

HTML5 Semântico: Estruturação focada em SEO e acessibilidade.

CSS3 Moderno:

Uso de Flexbox e Grid Layout para alinhamento preciso.

Responsividade Adaptativa: Implementação da função clamp() para fontes fluidas e aspect-ratio para evitar corte de imagens em dispositivos móveis.

Design baseado em tokens modernos (bordas arredondadas de 40px, paleta de luxo).

SEO Avançado:

Meta tags de descrição e Open Graph (para compartilhamento em redes sociais).

Dados estruturados JSON-LD para otimização de resultados de busca (Product Schema).

Integração WhatsApp: Link direto com mensagem personalizada pré-configurada.

📱 Responsividade

A página foi otimizada para garantir uma experiência premium em:

Desktops e Laptops.

Tablets (iPads).

Smartphones de diversos tamanhos (incluindo iPhone SE e modelos mais antigos/pequenos).
